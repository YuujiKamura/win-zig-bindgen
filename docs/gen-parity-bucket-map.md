# Gen Parity Bucket Map

Updated: 2026-03-08

## Purpose

`test-gen-parity` is now a real-generation comparator.  
This document defines the failure buckets and their owning issues.

For live counts, do not edit this file by hand. Regenerate from the current worktree with:

```powershell
pwsh -File scripts/classify-gen-parity-failures.ps1 -RepoRoot . -RunTest
```

Optional artifacts:

```powershell
pwsh -File scripts/classify-gen-parity-failures.ps1 `
  -RepoRoot . `
  -RunTest `
  -JsonOutPath docs/gen-parity-failures.json `
  -MarkdownOutPath docs/gen-parity-failures.md
```

Compare two snapshots:

```powershell
pwsh -File scripts/compare-gen-parity-failure-reports.ps1 `
  -BeforePath docs/gen-parity-before.json `
  -AfterPath docs/gen-parity-after.json `
  -MarkdownOutPath docs/gen-parity-diff.md
```

Split parent issues into case issues:

```powershell
pwsh -File scripts/sync-gen-parity-case-issues.ps1 `
  -RepoRoot . `
  -IssueRepo YuujiKamura/win-zig-bindgen `
  -ReportPath $env:TEMP\win-zig-bindgen-gen-parity-watch-live\latest-gen-parity.json `
  -BucketId CDFG `
  -ParentIssue 25
```

```powershell
pwsh -File scripts/sync-gen-parity-case-issues.ps1 `
  -RepoRoot . `
  -IssueRepo YuujiKamura/win-zig-bindgen `
  -ReportPath $env:TEMP\win-zig-bindgen-gen-parity-watch-live\latest-gen-parity.json `
  -BucketId D2 `
  -ParentIssue 26
```

Run continuous monitoring:

```powershell
pwsh -File scripts/watch-gen-parity-progress.ps1 `
  -RepoRoot . `
  -IntervalSeconds 60
```

Start background monitoring:

```powershell
pwsh -File scripts/start-gen-parity-watch.ps1 `
  -RepoRoot . `
  -IntervalSeconds 60
```

Start background monitoring and auto-comment `#23`:

```powershell
pwsh -File scripts/start-gen-parity-watch.ps1 `
  -RepoRoot . `
  -IntervalSeconds 60 `
  -EnableIssueComment `
  -IssueRepo YuujiKamura/win-zig-bindgen `

```

Check watch status:

```powershell
pwsh -File scripts/status-gen-parity-watch.ps1
```

Stop background monitoring:

```powershell

```

One-shot monitoring pass:

